Web Component

개인용 Web Component 입니다.
BEM CSS 방법론을 사용합니다.
구성요소를 감쌀때는 -wrap 접미사를 사용합니다.
ex) .table-wrap / .input-wrap / .list-wrap

# DOCTYPE

HTML5 DOCTYPE을 사용합니다. 

<!DOCTYPE html>
<html lang="ko">
	<head>
        
        Document Title
	</head>
</html>

# Theme

테마 사용시 body나 wrap에 theme 클래스를 추가합니다.
기본 글자 모양을 정의합니다. 

<body class="theme"></body>
<body class="theme theme--dark"></body>





.wrap {
	font-family:'Noto Sans KR','Roboto','serif';
	font-size:16px;
	line-height:1.6;
	color:#111;
}

# Reset

Reset Css 입니다.
필요한 부분은 추가하여 사용합니다.
Nomalize를 사용해도 됩니다. 

@charset "UTF-8";
html,body,div,dl,dt,dd,ul,ol,li,h1,h2,h3,h4,h5,h6,form,fieldset,p,input,textarea,select,button,th,td,blockquote {margin:0;padding:0}
body {-webkit-text-size-adjust:none}
ul,ol,li {list-style:none}
a {text-decoration:none}
img {max-width:100%;border:none;vertical-align:top}
button, input[type=button], input[type=submit] {-webkit-appearance:none;-webkit-border-radius:none}

.blind {visibility:hidden; overflow:hidden; position:absolute; top:0; left:0; width:1px; height:1px; font-size:0; line-height:0}
